# clojure-examples

Clojure hacking, courses and code examples

## Requirements

- OpenJDK 11
- Leiningen

## Generating Clojure Project

1. Generating project with Leiningen:

```bash
lein new app tutorial
```

2. Enter in generated folder

```bash
cd tutoral
```

### Running

Running app from terminal

```bash
lein run
```

### REPL

1. Run REPL

```bash
lein repl
```

2. Executing function

```bash
(-main)
```

3. Running commands:

```bash
(println "Hello World!")

(+ 2 3)
```
